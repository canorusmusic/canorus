** Installation Instructions **

1. Double-click the Canorus.pkg. This will run the installer, which installs Canorus and the necessary fonts.

2. Download and install Qt: http://www.trolltech.com/developer/downloads/qt/mac

3. Download and install Python: http://www.python.org/download/

4. Download, install, and run SimpleSynth: http://notahat.com/simplesynth (or another synth of your choice)

5. In /Applications/Utilities/Audio MIDI Setup, enable the "IAC Driver" (click "Device is online")

6. Run Canorus. In the Canorus menu, under Preferences, set the MIDI output device to "Bus 1".

7. To test Canorus, open the canorus-example.xml (File menu: Open...) file and press the play button.

Please report any problems to canorus-devel@lists.berlios.de
For more info, see the Canorus website, http://canorus.org
