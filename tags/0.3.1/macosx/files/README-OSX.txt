Canorus
Version 0.2.5

** Installation Instructions **

1. Double-click the Canorus.pkg. This will run the installer, which installs Canorus and the necessary fonts.

2. Download, install, and run SimpleSynth: http://pete.yandell.com/software/ (or another synth of your choice)

3. In /Applications/Utilities/Audio MIDI Setup, enable the "IAC Driver" (click "Device is online")

4. Run Canorus. In the Canorus menu, under Preferences, set the MIDI output device to "Bus 1".

5. To test Canorus, open the canorus-example.xml (File menu: Open...) file and press the play button.

6. Have fun!

Please report any problems to canorus-devel@lists.berlios.de
For more info, see the Canorus website, http://canorus.org
